# Projeto-NotbookLM
Esse repositorio foi criado para armazenar o meu projeto da DIO sobre IA.

O objetivo do projeto era criar um NotebookLM com um assunto introdutório financeiro, eu escolhi a bolsa de valores, é um assunto que eu não entendo, e achei que seria uma boa oportunidade para aprender.
Meu projeto não fala só sobre a bolsa de valores, mas de todo o caminho que se percorre até enteder esses conceitos, como educação financeira, juros, o que é um investimento.
Eu criei um "mapa" para tentar abranger todos esses assuntos e não ficar perdida no caminho.
