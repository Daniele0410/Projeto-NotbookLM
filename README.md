# Projeto-NotbookLM
Esse repositorio foi criado para armazenar o meu projeto da DIO sobre IA.

O objetivo do projeto era criar um NotebookLM com um assunto introdutório financeiro, eu escolhi a bolsa de valores, é um assunto que eu não entendo, e achei que seria uma boa oportunidade para aprender.
Meu projeto não fala só sobre a bolsa de valores, mas de todo o caminho que se percorre até enteder esses conceitos, como educação financeira, juros, o que é um investimento.
Eu criei um "mapa" para tentar abranger todos esses assuntos e não ficar perdida no caminho.

---------------------------------------------------------------------------------------------------------------------------------

Tema: Assunto financeiro introdutório 

* Como investir na bolsa de valores
 
   • Conhecimentos antes de investir 
       • Conhecimento básico financeiro
	       • Juros simples, juros compostos
	       • Porcentagem de lucro 
	       • Renda fixa, tesouro direto, CDBs, LCI/LCA, tesouro Selic etc.
	
   • Como funcionando o investimento na bolsa de valores

   • Riscos de investir na bolsa
      • Mitigar riscos
   
   •  O que é bolsa de valores
      • Conceito
         • Quais são os bolsas de valores
             • ON, PN, Units = B3
             • Mercado da bolsa.
                 • Mercado à vista, mercado primário, e etc.
   
   • Como escolher a bolsa certa
      • Menos riscos, mais lucros

   • Por onde começar a investir
 
   • Bolsa de valores nacional X internacional   
      • Comparar riscos/perdas
      • Comparar os ganhos

  ---------------------------------------------------------------------------------------------------------------------------------

As coisas liatadas não estão marcadas por uma ordem de importância, eu só fui escrevendo conforme eu fui elaborando ou achando necessário abordar o assunto.

Usando esse mapa eu consegui achar conteúdos muito relevantes. Ao todo eu consegui coletar 49 fontes. Foi difícil achar textos e vídeos que explicassem esse tema de uma forma clara e eficiente, obviamente eu não li e nem assisti todos os conteúdos, mas fiz questão de procurar conteúdos introdutórios ou para iniciantes. Eu acho que consegui trazer uma boa base de dados. Vou trazer arquivos de alguns links usados e de Prompts que criei para testa a IA.
